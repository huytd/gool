# GOOL - Google in a line

GOOL is an utility to help you search Google quickly right in the terminal. You can also have a quick look at the content of a result.

![](demo.gif)

## How to install?

Install it via `npm`:

```
npm install -g gool
```

## How to use

Search:

```
gool <search term>
```

Open a URL:

```
gool <result number>
```

See previous search result:

```
gool
```

Clear cache:

```
gool clean
```
